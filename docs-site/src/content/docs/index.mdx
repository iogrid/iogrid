## Four workloads, one daemon

iogrid is a single horizontally-integrated mesh. The same provider daemon, the
same coordinator, the same auth and billing, the same transparency layer — what
varies is the workload module.

<CardGrid stagger>
  <Card title="Bandwidth proxy" icon="seti:rss">
    Residential proxy at **$0.30–0.60/GB**, 10–30× under enterprise pricing.
    Geo-targeted, session-sticky, fully metered.
  </Card>
  <Card title="Docker compute" icon="seti:docker">
    Containers on idle home Linux + Mac providers. **$0.02–0.10/CPU-hour**
    with cgroup isolation and approved-registry-only image policy.
  </Card>
  <Card title="GPU / AI inference" icon="seti:cuda">

  </Card>
  <Card title="iOS build CI" icon="seti:apple">
    The wedge. **$0.04/Xcode-minute — 50% under GitHub Actions.** Ephemeral
    Tart VMs on real Macs. No 24-hour lease minimum like EC2 Mac.
  </Card>
</CardGrid>

## Why iogrid, not Bright Data / Honeygain / Salad

| | iogrid | Residential-proxy incumbents | Compute marketplaces |
|---|---|---|---|
| Workload mix | Bandwidth · Docker · GPU · iOS-build | Bandwidth only | Compute only |
| Provider sees what flows | **Live category labels, per-byte audit** | Opaque ToS | Limited dashboard |
| Provider payout | Cash · free VPN · $GRID · charity | Cash only | Cash or crypto |
| Customer pricing | $0.30/GB · $0.04/Xcode-min | $5–15/GB | n/a |
| Brand legacy | Built post-Hola, transparency-first | Hola scandal in lineage | Gamer-GPU niche |

The full master comparison lives in
[Competitive landscape](/concepts/overview/#competitive-landscape).
