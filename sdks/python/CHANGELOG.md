## [0.1.0] — TBD

### Added
- Initial public release of the `iogrid` Python package.
- Async-first client built on `httpx`.
- Ten methods covering workload submission, lifecycle, streaming, API
  keys, usage, and invoices — see `sdks/README.md` for the matrix.
- PyPI Trusted Publishing (OIDC) — every release is signed by GitHub's
  identity and the sigstore attestation is stored on PyPI.
